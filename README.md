# qualityProps

Modern Next.js website scaffold for a sports props platform.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Deploy
Deploy to Vercel by importing this repository and accepting defaults.

## Quick content edits
- Global layout/nav/footer: `src/components/layout/*`
- Home page: `src/app/page.tsx`
- Pricing/Picks/Results: `src/app/{pricing,picks,results}/page.tsx`
- Legal pages: `src/app/{terms,privacy}/page.tsx`
